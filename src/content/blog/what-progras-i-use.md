I use many programs to manage my workflow and here I will show you some of these programs.

### Code Editor and IDE's

* __Neovim__: Neovim is a highly extensible and improved version of the Vim text editor. It aims to provide better usability, scripting, and plugin support while maintaining compatibility with most Vim features.
* __VS Code__: Visual Studio Code (VSCode) is a free, open-source code editor developed by Microsoft. It supports multiple programming languages, offers features like debugging, syntax highlighting, intelligent code completion, snippets, and integrates with version control systems like Git.
* __Android Studio__: Android Studio is the official integrated development environment (IDE) for Android app development, created by Google. It provides tools for coding, debugging, testing, and building Android applications, and includes an emulator for testing apps on different devices.
* __IntelliJ__: IntelliJ IDEA is a powerful integrated development environment (IDE) developed by JetBrains, primarily for Java development. It offers advanced coding assistance, debugging, refactoring tools, and supports many other languages and frameworks through plugins.

### Terminal

* __Bash__: Bash (Bourne Again Shell) is a Unix shell and command language. It is widely used as the default shell on Linux and macOS systems, providing a command-line interface for interacting with the operating system and scripting tasks.
* __WezTerm__: WezTerm is a GPU-accelerated terminal emulator and multiplexer written in Rust. It offers features like support for multiple tabs, panes, and a modern, configurable interface, aiming to provide a fast and efficient terminal experience.

### Productivity

* __Notion__: Notion is a productivity and collaboration tool that combines note-taking, task management, databases, and wikis into a single application. It is designed to help individuals and teams organize their work, plan projects, and manage information efficiently.
* __ChatGPT__: ChatGPT is an AI developed by OpenAI for natural language processing and generating human-like text.

### Content Creation

* __OBS Studio__: OBS stands for Open Broadcaster Software, a free and open-source software for video recording and live streaming.
* __Kdenlive__: Kdenlive is a free and open-source video editing software for Linux, macOS, and Windows. It provides a wide range of features for editing videos, including multi-track editing, effects, transitions, and more.
* __Clipchamp__: Clipchamp is a web-based video editing platform that allows users to edit videos online. It offers features like trimming, cutting, adding text and audio, and applying effects and transitions directly through a web browser.
* __Audacity__: Audacity is a free, open-source audio editing software available for Windows, macOS, and Linux. It provides tools for recording and editing audio, including features like multi-track editing, effects, noise reduction, and more.
